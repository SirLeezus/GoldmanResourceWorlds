## About

Resource world system leveraging the Java Paper Spigot API. Noteworthy features encompass automatic world resets and a GUI menu for teleporting to resource worlds.

* On Linux or macOS: `./gradlew build`
* On Windows: `gradlew build`

## Minecraft Version Supported

[Paper](https://papermc.io/software/paper) `1.20.X`
